Huidige GDI-bouwsteen
DigiToegankelijk is de Nederlandse naam voor de standaard waarmee overheidsorganisaties hun websites en apps toegankelijk moeten maken voor alle burgers. DigiToegankelijk is gebaseerd op een internationale toegankelijkheidsstandaard: de Web Content Accessibility Guidelines (WCAG) <aanpassen tov ADO30: is geen standaard maar een aantal toezichtsafspraken en dashboard>
