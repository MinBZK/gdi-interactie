Faciliteren Overheidsbrede loketten ()
De overheid kan overheidsbrede loketten faciliteren. Dit zijn lokketten waar namens meerdere overheidsorganisaties op een fysieke plek basis publiekdienstverlening aangeboden wordt aan Iedereen* die er digitaal niet uitkomt. Waar toepasselijk wordt Iedereen* zelfredzaam gemaakt. Waar mogelijk wordt “assisted service” geboden. Uitgangspunt is dat verdere taakafhandeling op dezelfde manier verloopt als bij het digitale defaultkanaal.
