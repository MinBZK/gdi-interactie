Begrippen > bouwsteen
DefinitieDe term GDI-bouwsteen is gedefinieerd in het BDO22, art. 1, lid c. Het is geen strikt architectuurbegrip. bnn Het kan met enige voorzichtigheid geassocieerd worden met het in Togaf gedefinieerde Architectural Building Block. Het subtype “voorziening” kan geassocieerd worden met Solution Building Block. Zie TOGAF Standard v10 — Introduction and Core Concepts § 3.6.
